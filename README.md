# VA Portfolio

A personal portfolio site built for virtual assistant work — showcasing services,
experience, and how to get in touch. Built with plain HTML, CSS, and JavaScript, no
framework or build step.

**Live:** [joyce-va-portfolio.netlify.app](https://joyce-va-portfolio.netlify.app/)

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Running Locally

No build tools required — clone the repo and open `index.html` directly in a browser,
or serve it with any static file server:

```bash
git clone https://github.com/0seme/va-portfolio.git
cd va-portfolio
python3 -m http.server
```
